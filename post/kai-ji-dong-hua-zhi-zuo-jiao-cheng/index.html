开机动画制作教程

2022-01-01 3 min read # 红米k30p玩机手册

看了网上的开机动画制作教程，大多都是参差不齐，含糊不清，那么本文就来给大家从头到尾详细讲解安卓开机动画。

desc.txt中代码如下：
1080 1920 20
p 1 0 intro #ffffff
p 0 0 loop #ffffff
c 1 40 outro #ffffff

补充一点desc.txt最后一行必须回车，至于回几个车你开心就好。。。
第一行：1080 1920 20
第一个与第二个参数1080(宽)、1920(高)即分辨率，看了很多教程都把这个分辨率说成是动画图片的分辨率，其实并不然，只要这个分辨率和你手机分辨率对上了不管动画图片分辨率是多少它都会自适应填充。第三个20(帧率即播放速度)，每秒播放20张图片。
第二行：p 1 0 intro #ffffff
第一个参数p(part)，即播放部分，就是开机好了动画还没播放完的话就终止播放，第二个参数1(播放次数，0则无限循环)，即播放一次，第三个参数0(延迟)，即播放完后延迟0s播放下部分，第四个参数intro(路径)，即这部分动画图片在压缩包中的路径(bootanimation.zip/intro)，第五个参数#ffffff(背景颜色)，即动画图片的背景颜色，将desc.txt中的分辨率写成小于手机分辨率这时动画图片就不会填满屏幕，多余的部分就会是第五个参数设置的颜色(第三行同上)
第四行：c 1 40 outro #fffff
第一个参数c(continue)，即完整播放，也就是说开机好了动画还没播放完的话则继续播放，如果你将第二个参数写成0，并不会一直无限播放下去，系统会默认为p 0也就是循环播放到开机结束，后面的参数同上。
处理gif什么的这里就不说了用gif助手酷安就有，压缩普通的压缩会导致黑屏可以用MT管理器以储存模式压缩，好了本文就到此结束了。
